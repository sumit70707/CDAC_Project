# TrueMe - Scalable E-Commerce Microservices Platform

TrueMe is a production-grade E-Commerce application built using a modern **Spring Boot Microservices Architecture** with a React frontend.

The system is designed with scalability, fault tolerance, security, and cloud deployment in mind.

# üîÑ Order Placement Flow

1. User logs in via React App
2. JWT token generated by Auth Service
3. Request goes through API Gateway
4. Order Service validates cart
5. Order Service calls Payment Service
6. Payment Service processes payment
7. Kafka event published
8. Notification Service consumes event
9. Email notification sent
